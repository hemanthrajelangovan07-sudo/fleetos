# 🚛 FleetOS

A modern, AI-powered fleet management dashboard built with React. FleetOS provides a centralized interface for assigning vehicles, tracking operations, predicting maintenance, and reducing fleet overhead — all in one place.

---

## 📸 Overview

FleetOS is a single-file React component (`FleetOS.jsx`) that serves as a fully functional fleet management dashboard. It is designed to help fleet managers, dispatchers, and operations teams manage their vehicles and drivers efficiently from a clean, intuitive UI.

---

## ✨ Features

- **Vehicle Assignment** — Drag-and-drop task and vehicle assignment to reduce coordination time
- **Predictive Maintenance** — Real-time diagnostics and alerts to prevent breakdowns before they happen
- **Driver Management** — Invite and manage drivers, admins, and dispatchers with role-based access
- **Fleet Analytics** — Reduce fuel waste, downtime, and admin overhead with powerful performance insights
- **Digital Documentation** — Go fully paperless with vehicle docs, driver logs, and audit-ready records
- **Real-Time Dashboard** — Monitor your entire fleet's health and status at a glance

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher)
- npm or yarn

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/hemanthrajelangovan07-sudo/fleetos.git
   cd fleetos
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
